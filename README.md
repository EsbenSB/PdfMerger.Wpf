# PdfMerger.Wpf

A simple WPF application written in C# that allows users to merge multiple PDF files into a single PDF in a custom order.

## Features
- Select multiple PDF files
- Reorder files using move up / move down
- Merge PDFs in the selected order
- Preserves original PDF layout

## Technologies
- C#
- .NET 8 (Windows)
- WPF
- PdfSharp 6.2.4

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio 2022 or newer
3. Restore NuGet packages
4. Run the project (F5)

## Requirements
- Windows
- .NET 8 SDK
- Visual Studio 2022+
